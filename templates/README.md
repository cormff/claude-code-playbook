# Vault Skeleton Templates

Minimal starter files for bootstrapping a new Obsidian vault along the conventions described in [`skill-tree/layer-0-foundation/vault-format`](../skill-tree/layer-0-foundation/vault-format/SKILL.md).

- **`Hub.md.template`** — copy to `_Hub.md` at the vault root, and to `<Area>_Hub.md` for each top-level area.
- **`Project-Note.md.template`** — copy as the main note for any new project or topic.
- **`Status.md.template`** — copy to `Dev_Tracking/Status.md` for any project using the [`project-orchestrator`](../skill-tree/layer-2-orchestrator/project-orchestrator/SKILL.md) pattern — this is the "where we left off" file.

Strip the `.template` suffix and fill in the placeholders (anything in `{curly braces}`) when you copy one of these in.
